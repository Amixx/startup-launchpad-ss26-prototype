# startup-launchpad-ss26-prototype

## GitHub Pages previews

`main` is deployed to the root GitHub Pages URL:

```text
https://amixx.github.io/startup-launchpad-ss26-prototype/
```

Branches under `preview/` are deployed under `/previews/`.
Example:

```text
preview/christian-feedback -> /previews/christian-feedback/
```

The preview index is available at:

```text
https://amixx.github.io/startup-launchpad-ss26-prototype/previews/
```

Every Pages deploy rebuilds the root `main` version and all matching preview
branches into one artifact, so pushing one preview branch does not erase the
others.
